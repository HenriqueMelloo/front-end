Ao digitar ! e pressionar Tab ou Enter, 
o editor expande atutomáticamente para a estrutura básica de um documento HTML. 
Aqui está como isso funcionar:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>
    
</body>
</html>
